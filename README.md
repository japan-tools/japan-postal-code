# Japan-Postal-Code
 全日本郵便番号のJsonデータです。  
 
 ## 説明
 このJsonデータは商品開発向けに圧縮されています、Jsonのファイル名は郵便番号先頭の3桁数で命名されています。  
 
 データの中身は下記の感じです。
 ```json
 {"0010000":[1,"札幌市北区",""],"0010010":[1,"札幌市北区","北十条西"],"0010011":[1,"札幌市北区","北十一条西"],"0010012":[1,"札幌市北区","北十二条西"],......
 ```
